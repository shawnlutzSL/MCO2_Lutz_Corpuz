# MCO2_Lutz_Corpuz

Best to compile via IntelliJ for ease of access.

**Controls of the game:**
W: Move Up
A: Move Down
S: Move Left
D: Move Right
Space: View Inventory
Left Click: To interact with any amenity

**Important Notes**
- The shopper may only interact with any amenity within a 1 tile radius/range.
- When opening a display, it is automatically put in "Taking a product" mode.
